 # Personal Portfolio Website

## Overview
This is my personal portfolio project built using frontend and Spring Boot backend.

## Features
- Responsive portfolio design
- About section
- Skills section
- Projects showcase
- Contact form with backend support

## Technologies Used
Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Java
- Spring Boot
- REST APIs

## Project Setup

### Run Backend
bash
mvn spring-boot:run


### Run Frontend
Open index.html in browser
